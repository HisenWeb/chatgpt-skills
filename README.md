# ChatGPT Handoff Skills

本目录用于存放 ChatGPT 新窗口交接相关 Skill 与模板。

## 文件结构

```text
.
├─ README.md
├─ SKILL_INDEX.md
├─ skills/
│  ├─ 04-handoff-regular.md
│  └─ 04b-handoff-distillation.md
└─ templates/
   ├─ handoff-regular-current-window.md
   ├─ handoff-distill-from-chat-export.md
   ├─ new-window-read-handoff-file.md
   └─ new-window-read-handoff-clipboard.md
```

## 使用原则

1. 先读 `SKILL_INDEX.md`。
2. 根据任务选择一个相关 Skill。
3. 常规换窗口用 `04-handoff-regular.md`。
4. 完整旧聊天记录进入中转窗口蒸馏时，用 `04b-handoff-distillation.md`。
5. 模板只负责启动任务，不替代 Skill 正文。
